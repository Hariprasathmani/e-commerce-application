# 🏍️ MotoShowroom - Complete E-Commerce Platform

[![Status](https://img.shields.io/badge/Status-Production%20Ready-green)]() 
[![Version](https://img.shields.io/badge/Version-1.0.0-blue)]()
[![License](https://img.shields.io/badge/License-MIT-orange)]()

A modern, full-stack e-commerce platform for motorcycles and accessories. Built with vanilla JavaScript, Python Flask, and MongoDB.

## ⚡ Quick Start (2 minutes)

### Windows
```batch
cd e_commerce_app
setup.bat
```

### Linux/macOS
```bash
cd e_commerce_app
chmod +x setup.sh
./setup.sh
```

Then open **`http://127.0.0.1:5500`** in your browser.

## ✨ Key Features

### 🛍️ Shopping
- Browse 1000+ motorcycles and gear
- Advanced search and filtering
- Detailed product pages
- Shopping cart with persistent storage
- Secure checkout process

### 👤 Authentication
- User registration & login
- Secure password hashing (bcrypt)
- JWT authentication
- User profiles

### ⚙️ Admin
- Admin dashboard with statistics
- Product management
- Order tracking
